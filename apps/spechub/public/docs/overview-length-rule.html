📏 Overview 字數規則

Datasheet 封面排版邏輯，給 PM / MKT 參考

📐 為什麼沒有固定字數上限？

Cover 頁的 Overview 跟 Features 共用一塊空間。Features 越多，Overview 能放的就越少。

Hero + Hardware Image
Overview ← 彈性 吃剩下的空間
Features（最多 320pt）

Features 先佔，Overview 拿剩下的。

同一頁的空間是固定的，所以你寫越多 features，留給 overview 的位置自然越少 — 反過來也一樣。

🚨 超出會怎樣？

  1. Dashboard 對應 OV 欄位顯示紅點
  2. Product 頁出現紅色警告橫幅
  3. 不會擋 PDF 生成，但建議先預覽確認版面
  4. 看過覺得 OK → 按「Mark as Reviewed OK」紅燈消失
  5. 之後若有人再改 Overview / Features，紅燈會自動回來再提醒

💡 重點提醒

  • 每個語言獨立判斷 — 英文過不代表日文也過
  • 紅燈是提示不是禁令 — 確認過版面 OK 就可以 Mark
  • 順序：寫完先看 Dashboard → 紅點就預覽確認 → 沒事就 Mark OK
