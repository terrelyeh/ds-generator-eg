💬 這個 Widget 怎麼用

底下這幾步，就是其他部門把它放上自己網站後，使用者會經歷的流程。

  1. 看右下角 ↘ 的藍色聊天按鈕 —— 點它打開聊天面板。
  2. 用自然語言直接發問，例如：「哪些 AP 支援 WiFi 7？」「ECW536 的規格是什麼？」「推薦適合飯店的網路方案」。
  3. 答案會即時串流出來，並附上來源引用；可繼續追問、或左上選單切換模型／角色。
  4. 關閉面板後再點按鈕可隨時回來；對話歷史保留在你的瀏覽器。
